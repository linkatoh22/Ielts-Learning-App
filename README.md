## 2. Installation:
**1. Clone repo:**
```bash
git clone https://github.com/linkatoh22/PetStore-Web.git
```
**2. Install Frontend:**
```bash
cd frontend
npm install
npm run dev
```
**3. Install Backend:**
```bash
cd backend
npm install
npm run dev
```
